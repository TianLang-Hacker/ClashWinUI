# 🚀 ClashWinUI

ClashWinUI 是一款基於 WinUI 3 構建的現代化本地網路流量管理與路由控制台。本專案採用前、後端徹底解耦的架構設計，允許使用者自訂載入第三方網路處理核心（如 Mihomo），為您提供透明、高效且安全的網路規則設定體驗。

[核心功能](#-核心功能) • [安裝指南](#-安裝指南) • [使用文件](#-使用文件)

---

## ✨ 核心功能

* **網路流量監控**：即時分析與接管本地網路請求，提供直觀的流量流向檢視。
* **本地網路流分發**：支援建構在地化的網路轉發規則，精準控制各個應用程式的數據通路。
* **規則路由策略**：基於 YAML 設定的強大路由引擎，實現靈活的分流與網路節點管理。
* **現代化原生體驗**：完全遵循 Windows 11 的 Fluent Design 設計規範，深度支援 Mica/Acrylic 材質與系統深色模式無縫切換。
* **高度自由的核心管理**：純前端架構，支援使用者自訂匯入及切換外部網路核心執行檔，實現真正的「設定與介面」分離。

## 📦 安裝指南

如果您在 Release 頁面下載的 `.msix` 安裝套件由於**本地自簽名憑證**的原因提示無法直接安裝，請按照以下步驟匯入信任憑證即可解決：

1. **擷取憑證資訊**：右鍵點擊下載的 `.msix` 安裝套件，選擇 **「內容」**。
2. **檢視簽章**：切換到 **「數位簽章」** 索引標籤，在清單中選取簽章者（如「內嵌簽章」），點擊 **「詳細資料」**。
3. **安裝憑證**：在彈出的視窗中點擊 **「檢視憑證」**，隨後點擊 **「安裝憑證」**。
4. **選擇存放位置**：在憑證匯入精靈中，將存放位置選擇為 **「本機電腦」**，點擊「下一步」。
5. **指定憑證目錄**：選擇 **「將所有憑證放入以下的存放區」**，點擊「瀏覽」，選擇 **「受信任的根憑證授權單位」**。
6. **完成安裝**：點擊「下一步」直到「完成」。系統提示匯入成功後，即可雙擊 `.msix` 檔案正常安裝應用程式。
